# 贪吃蛇游戏

一个功能完整的贪吃蛇游戏网站，包含速度选择、障碍物系统和特殊食物等特性。

## 功能特性

- **速度选择系统**：5个速度档位（慢速、中速、快速、超速、极速）
- **障碍物系统**：从第6个食物开始出现，使用BFS算法确保可玩性
- **特殊食物系统**：红色加速食物和蓝色减速食物
- **游戏控制**：开始、暂停、重置功能
- **分数系统**：当前得分和最高分记录

## 技术实现

- 使用Canvas绘制游戏画面
- 实现BFS路径检测算法确保游戏可玩性
- 响应式设计，现代简约风格
- 本地存储最高分记录

## 操作指南

1. 选择游戏速度并点击确认速度
2. 点击开始游戏开始
3. 使用方向键控制蛇的移动
4. 吃到食物得分，避免撞墙和撞到自己
5. 从第6个食物开始出现障碍物
6. 从第5个食物开始出现特殊食物（1/8概率）
